# negativas: _analisando negações sentenciais_


Este é o repositório da ferramenta `negativas`, Notebook que tem como objetivo a implementação de um código de busca e classificação de negações sentenciais em arquivos de entrevistas transcritas da amostra _Deslocamentos_, do Banco de Dados Falares Sergipanos¹.

A ferramenta encontra-se no arquivo `negativas.ipynb`, e está organizada da seguinte forma: a **parte 1** trata da instalação dos componentes necessários para a execução do código; a **parte 2** traz a implementação das funções que serão utilizadas pela ferramenta; o código responsável pela análise (busca e classificação) encontra-se na **parte 3**; por fim, a **parte 4** trata do armazenamento dos dados em forma de planilhas.

---

1. FREITAG, R. M. Ko. Banco de dados falares sergipanos. **Working Papers em Linguística**, v. 14, n. 2, p. 156-164, 2013. DOI: https://doi.org/10.5007/1984-8420.2013v14n2p156
